# 🌐 Portfolio React App

This is my personal portfolio website built with **React**, styled using **Tailwind CSS**, animated with **Framer Motion / GSAP**, and deployed on **GitHub Pages**.

---

## 🚀 Technologies Used

* **React** (Create React App)
* **Tailwind CSS** (modern styling)
* **Framer Motion / GSAP** (animations & transitions)
* **GitHub Pages** (deployment)
* **Node.js** & **npm**
* **GitHub Actions** (automated deployment)

---

## ✨ Features

* 🎨 Responsive design (mobile, tablet, desktop friendly)
* 🌙 Dark / Light mode toggle
* ⚡ Smooth animations (GSAP / Framer Motion)
* 🧑‍💻 Sections: About, Skills, Experience, Projects, Contact
* 📬 Integrated "Back to Top" navigation
* 📧 Contact form integrated with EmailJS
* 🔗 Live deployment with GitHub Pages

---

## 📂 Project Setup

1. **Clone the repository**

```bash
git clone https://github.com/Samandeep246/portfolio-react.git
cd portfolio-react
```

2. **Install dependencies**

```bash
npm install
```

3. **Run the app locally**

```bash
npm start
```

* The app will open in your browser at: `http://localhost:3000`

---

## 🛠️ Build & Deployment

1. **Build the project**

```bash
npm run build
```

2. **Deploy to GitHub Pages**

```bash
npm run deploy
```

* Once deployed, your site will be live at: [https://samandeep246.github.io/portfolio-react](https://samandeep246.github.io/portfolio-react)
